# My Saves

A thoughtful content library for the things worth keeping.

## What it does

- Saves links from X/Twitter, Substack, email newsletters, Instagram, YouTube, and the wider web
- Detects the source automatically
- Suggests a category from the title or excerpt
- Creates a concise summary for every save
- Keeps the original link one click away
- Supports search, category filters, favorites, recent saves, and deletion
- Stores your library in your browser so it remains private to your device
- Works on desktop and mobile

## Design

The interface uses a 50/40/10-inspired palette: warm white for breathing room, cobalt blue for structure, and butter yellow for energy and focus.
